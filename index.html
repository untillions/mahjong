The Suits

Characters Suit

萬 (wàn: sounds like “Juan”)

In the characters suit, the character 萬 (wàn, meaning ten thousand) appears on the tile below each number. To read the tile, read the number character followed by the suit name.

Example: read qī wàn (“chee Juan”)

Bamboos Suit

索 (suǒ: sounds like “tsea-ow”)

In the bamboos suit, the first tile usually depicts a bird rather than a single bamboo stick. The other tiles can be distinguished by the number of bamboo sticks.

Example: read èr suǒ (“er tsea-ow”)

Circles Suit

筒 (tǒng: sounds like “tong”)

In the circles suit, all tiles can be distinguished by the number of circles and are read in the same way as the other two suits.

Example: read wǔ tǒng (“oo tong”)
